#  Мурманск - Достопримечательности

Веб-приложение на Django для публикации и просмотра достопримечательностей Мурманска 
и Мурманской области с возможностью оставлять отзывы, оценивать места и загружать 
фотографии.

Данный проект включает в себя 3 основных раздела:

- Работа с достопримечательностями
- Личный кабинет пользователя
- Работа с отзывами

Виртуальное окружение используемое для проекта: .venv

### Настройка медиа-файлов
# Создайте папки для загрузки изображений:

```bash
mkdir -p media/sights
mkdir -p media/sights/gallery
mkdir -p media/users
```

1) Затем установите зависимости из файла requirements.txt

```bash
  pip install -r requirements.txt
```

2) Заполните .env файл согласно .env_sample
3) Создайте Базу Данный при помощи команды:

```bash
  python manage.py ccdb
```

4) Создайте миграции при помощи

```bash
  python manage.py makemigrations
```

5) Проведите миграции при помощи

```bash
  python manage.py migrate
```

6) Выполните команду для создания пользователей с разным урровнем доступа

```bash
  python manage.py ccsu
```

7) Выполните команду для заполнения БД используя фикстуры

```bash
  python loaddata sights_fixture.json
  python loaddata reviews_fixture.json
```

8) Выполните команду для запуска Redis

```bash
  redis-server
```

9) Откройте второе окно терминала

```bash
  python manage.py runserver
```

Модели используемые в проекте:

# Sights с полями:

- name - Название;
- category - Категория;
- short_description - Краткое описание;
- full_description - Полное описание;
- address - Адрес;
- website - Официальный сайт;
- image - Изображение;
- opening_hours - Часы работы;
- ticket_price - Стоимость билета;
- views_count - Просмотры;
- sight_instance - Карта;
- SightPhoto - Фотогалерея;

# Reviews с полями:

- sight - Достопримечательность;
- author - Автор;
- title - Заголовок;
- content - Содержимое;
- rating - Оценка;
- created - Дата создания;
- slug - URL;
- sign_of_review - Активен

# Users с полями:

- email - Эл. почта;
- role - Статус пользователя;
- first_name - Имя;
- last_name - Фамилия;
- avatar - Аватар;
- phone - Номер телефона;
- telegram - Аккаунт телеграм;
- is_active - Статус аккаунта;

## Возможности

### Для всех пользователей
- Просмотр списка достопримечательностей с пагинацией
- Просмотр подробной информации о каждом месте
- Отображение местоположения на Яндекс.Картах
- Просмотр фотогалереи достопримечательностей
- Просмотр рейтинга и отзывов
- Поиск достопримечательностей и категорий

### Для авторизованных пользователей
- Написание отзывов с оценкой от 1 до 5
- Редактирование и удаление своих отзывов
- Редактирование личного профиля (аватар, имя, телефон, telegram)
- Смена пароля

### Для модераторов
- Добавление, редактирование и удаление достопримечательностей
- Управление категориями
- Активация/деактивация отзывов пользователей
- Просмотр списка всех пользователей

### Для администраторов
- Полный доступ ко всем функциям
- Удаление любых отзывов
- Управление пользователями
- Доступ к стандартной админ-панели Django
